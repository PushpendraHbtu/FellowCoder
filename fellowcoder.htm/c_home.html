Introduction -

C language was developed by DENID RITHCIE in 1972 in Bell Laboritries . C is a mid level structured programming language. Mid level refers to between low level language & high level language.

Features of C language -

1). C has a good collection of built in functions annd operators, which can be used to write complex programs.

2). The 'C' compiler has capability of assembly language with the features of high level language and therefore it has it is goog for writing system software and business package.

3). Programs written in 'C' language are efficient and fast.

4). 'C' is highly portable, which means that programs written for one computer can run on another.

4). There are total 32 keywords in ANSI 'C'.

5). 'c' language is well suited for writing structured programming and business package.

C as a mother language

Most of the compilers; like JVM , KERNAL are developed using C language , so C language is cosidered as mother of all programming languages.

It has features like array,strings,function,file handling
