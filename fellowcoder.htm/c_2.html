PRINTF & SCANF

1). printf () :-

The printf() functon is used to print output on the screen.It is an formatted output function.Its defintion is written in the header file stdio.h.

The printf () function is used to print the character ,string,floating point,integer,octal and hexadecimal values onto the output screen. It is used with formatted character as %d,%c,%s etc.

EXAMPLE of printf () :-

#include <stdio.h>

void main()

{

printf("My name is fellow coder.");

}

2). scanf () :-

The scanf () function is used to take or read input from the keyboard and writes the result into the arguments given.

It is a formetted function,which is defined in header file named stdio.h.

& (ampersand)operator is used to take input.

EXAMPLE of scanf () :-

#include <stdio.h>

void main()

{

int a;

printf("Enter a number");

scanf("%d",&a);

printf("Value of a is %d",a);

}
